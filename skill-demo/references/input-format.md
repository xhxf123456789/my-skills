## 概览
本文档定义了 Skill 输入数据的标准格式，确保脚本能够正确解析输入内容。

## 格式定义

### 文本输入格式
- **类型**：字符串
- **编码**：UTF-8
- **最大长度**：10000 字符
- **允许字符**：字母、数字、标点符号、空格、换行符

### 格式类型参数
- **可选值**：
  - `uppercase`: 转换为大写
  - `lowercase`: 转换为小写
  - `title`: 转换为标题格式（首字母大写）

## 验证规则

### 必需验证
1. **非空检查**：输入文本不能为空字符串
2. **长度检查**：输入文本长度 ≤ 10000 字符
3. **编码检查**：输入必须是有效的 UTF-8 字符串

### 格式参数验证
- 必须是以下值之一：`uppercase`、`lowercase`、`title`
- 大小写敏感（必须完全匹配）

## 示例

### 示例1：大写转换
**输入**：
```
--input "hello world" --format uppercase
```

**输出**：
```
HELLO WORLD
```

### 示例2：标题格式
**输入**：
```
--input "the quick brown fox" --format title
```

**输出**：
```
The Quick Brown Fox
```

### 示例3：小写转换
**输入**：
```
--input "IMPORTANT NOTICE" --format lowercase
```

**输出**：
```
important notice
```

## 错误处理
当输入不符合规范时，脚本将返回错误信息：
- 输入为空：`错误: 输入文本不能为空`
- 格式类型错误：`错误: 不支持的格式类型: xxx，请使用 uppercase/lowercase/title`
- 编码错误：`错误: 输入包含无效字符`
